About the System

This project is an intelligent Vehicle Number Plate Detection and Recognition system designed to process vehicle data from multiple input sources. It uses advanced deep learning and computer vision techniques to accurately detect number plates and extract alphanumeric information in a fully automated manner.

Multi-Source Input Support

The system provides four flexible input modes to suit different real-world scenarios:
• Upload Image: Detect and recognize number plates from a single image file.
• Upload Video: Process recorded videos and extract number plate details frame by frame.
• Live Camera: Perform real-time detection using a connected camera for instant results.
• Capture Image: Capture a frame directly from the camera and analyze it immediately.

High Precision Detection

Robust deep learning models accurately detect number plates under varying lighting conditions, occlusions, and viewing angles.

Multi-Source Input

Seamlessly processes uploaded images, recorded videos, live camera feeds, and captured frames within a unified interface.

Optimized Real-Time Inference
